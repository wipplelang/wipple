# How to use `;`

The `;` operator lets you create a tuple (a list of values):

---
id: ny7gAWlC
type: code
lint: false
---

stuff : 1 ; "hi" ; True

---
id: LE0OIzS5
type: text
locked: true
---

Unlike a `list`, a tuple is fixed in size and each value can be a different type. To get the values back out of a tuple, use the `;` operator on the left-hand side of a variable assignment:

---
id: eS_1aST2
type: code
---

stuff : 1 ; "hi" ; True
number ; text ; bool : stuff
show number
show text
show bool

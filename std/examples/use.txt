# How to use `use`

`use` allows you to import third-party libraries. For example, the Wipple Playground includes the `turtle` library for making drawings:

---
id: 2pUKa3ef
type: code
---

use "turtle"

with-turtle {
  color `blue`
  forward (50 pixels)
}

If you want to share your own code, you can upload it to [GitHub Gist](https://gist.github.com) and use `wipple.dev/gist`:

---

type: code
---

use "https://wipple.dev/gist/f81ed1634d9ad44758a78f340a349ccf/say-hello.wpl"

say-hello ()
